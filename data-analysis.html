Data Analysis

This course provides a structured introduction to data science and machine learning, covering Python programming, data manipulation with Pandas, and data processing techniques, including cleaning and visualization. Participants will explore dimensionality reduction methods like PCA, UMAP, and t-SNE before delving into supervised and unsupervised learning using Scikit-Learn. By the end, they will be equipped with essential skills to preprocess data, visualize insights, and build machine learning models efficiently.

TP1 : Python Introduction | Cours - TP - Correction 

TP2 : Pandas Basics | Cours - Data cours - TP - Data TP - Correction

TP3 : Data processing & Data visualisation | Cours - TP (part1) - Data1 - TP(part2) - Data2 - Correction 1 - Correction 2

TP4 : Unsupervised Learning: Dimension reduction & clustering  | Cours - TP

TP5 : Supervised Learning | Cours - TP - Data
