Deep Learning Project

Some videos for background :

You can start by looking at But what is a neural network REALLY by Algorithmic Simplicity. To go further the whole 3b1b's Neural networks playlist is a really good starting point.

The first two channels focus heavily on mathematics, so if you're seeking a perspective more aligned with computer science, consider checking out Sebastien Lague's video : How to Create a Neural Network (and Train it to Identify Doodles). 

Before jumping into Pytorch and Tensorflow, you can try to build your own neural network from scratch to see if you fully understand the basics of backpropagation and neural network.
